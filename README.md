# README.MD

ZYFTP文件管理

- **简介** ：ZYFTP是一个简单的FTP文件管理器,允许您的安卓设备运行文件传输服务器；
- **功能** ：自定义用户名、密码和端口号、支持匿名访问；
- **用法** ：在Windows资源管理器地址栏里面输入显示的ftp地址即可访问相应的手机内存。
